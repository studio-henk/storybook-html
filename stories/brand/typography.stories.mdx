# Typography
**Font:** ff-scala-sans-pro

**Weights:** 300(light), 400(regular)

HENK uses the _Scala Sans_ font family.
Designed by _Martin Majoor_ in 1993. Published by FontFont.

HENK's body text is set in _Light_ with ```font-size: 16px;``` and ```line-height:28px;``` (1.75).
