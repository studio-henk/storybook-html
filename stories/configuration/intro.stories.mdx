# Configuration

HENK's main stylesheet *toolkit.css* loads the tokens files first and _config.css uses the values:

```css
/*tokens*/
@import url(tokens/_tokens-colors.css);
@import url(tokens/_tokens-size-font.css);
@import url(tokens/_tokens-size-spacings.css);

@import "_config.css";
```
